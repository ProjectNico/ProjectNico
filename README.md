# Project Nico

------
一个轻量级网络混淆代理，基于 SOCKS5 协议。


> * 只专注于混淆，用最简单高效的混淆算法达到目的
> * 不会放大传输流量，传输流量更少更快，占用资源更少（内存占用1M左右，CPU 占用 0.1% 左右）
> * 用Go写的，全是坑
> * 加密算法还在研究 💊
